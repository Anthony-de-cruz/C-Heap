# C-Heap
An implementation of a heap, along with functions `malloc()` and `free()`, in C.
### Build
Build with:
```sh
cmake -B ./build
cmake --build ./build
```
![The build process](doc/readme_image_build.png)

Run the example with:
```sh
./build/C-Heap
```
