# C-Heap
An implementation of a heap, along with functions `malloc()` and `free()`, in C.

### Build
Build with:
```sh
cmake -B ./build
cmake --build ./build
```
![[readme_image_build.png]]

Run with:
```sh
./build/C-Heap
```
